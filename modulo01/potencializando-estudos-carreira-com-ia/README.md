## 🛠️ Como Estudar e Progredir?

1. **Separe 1 hora diária**: Consistência é melhor que intensidade. Estudar 1 hora todos os dias trará mais resultados do que tentar estudar 7 horas em um único dia.
2. **Utilize as Anotações**: Para cada semana de estudos, copie o modelo `anotacoes/semana-01.md` mudando o nome para `semana-02.md`, `semana-03.md`, etc., e documente suas dúvidas, códigos praticados e insights.
3. **Pratique Codificando**: No backend, o aprendizado se consolida no terminal. Crie pequenos scripts e teste o comportamento do seu código a cada dia.
